# LLM Autocomplete 

## Setup

### Frontend (Vue.js)
- `yarn`
- `yarn dev --host`

### Backend (Python Flask)
- `cd server`
- `cp .env.sample .env`
- Fill in `.env`
- `python -m venv venv`
- `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
- `pip install -r requirements.txt`
- `python main.py`

### Prompt Testing (Promptfoo + Llama 3 + Groq)
- `yarn`
- `yarn ptest`
- `yarn view`
